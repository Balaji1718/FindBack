# FindBack

## Overview
FindBack is an Android application designed to help users find lost items and report found items within specific institutions. It features role-based access for students and administrators, institution selection, and real-time notifications.

## Features
- **Institution Selection**: Choose your organization to see relevant items.
- **Item Reporting**: Post lost or found items with descriptions and images.
- **Search & Filter**: Easily find items based on keywords.
- **Admin Dashboard**: Manage claims and verify ownership.
- **Dark Mode Support**: Fully compatible with system-wide dark themes.
- **Push Notifications**: Stay updated on your reported items.

## Setup Instructions
1. Clone the repository.
2. Open the project in Android Studio.
3. Connect your Firebase project and add the `google-services.json` file to the `app/` directory.
4. Sync the project with Gradle files.
5. Build and run the app on an emulator or physical device.

## Usage
- Upon first launch, select your institution.
- Log in or sign up to start posting and viewing items.
- Use the search bar to look for specific lost property.

## Notes
- Ensure you have a stable internet connection for Firebase features.
- This project uses Material Design 3 components.
